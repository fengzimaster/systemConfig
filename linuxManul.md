##Linux manul
### common command
    * `sudo !!` 执行上次操作
    * `history` 历史操作
    * `ctrl + R` 搜索历史命令
    * `updatedb` 更新locate，查询更快
    * `locate` 查找

### SSH command
    * `ssh-keygen -t rsa` 产生密钥
    * `cp id_rsa.pub authorized_keys` 登录检测密钥
    * `scp servername: location` 复制文件

### 查看机器信息
    * `netstat -net`
    * `ifconfig` ip地址
    * `free -m` 内存大小
    * `uname -a` 版本信息
